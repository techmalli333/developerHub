-------------------------------------
COMMANDS FOR INITILIZING PROJECT
-------------------------------------

npm init -y ==> for initilazation of project with default values

npm i express mongoose jsonwebtoken cors
    ==> express for backend
        mongoose for connecting mongo database
        jsonwebtoken for login purpose
        cors(cross origin policess) for to acces backend routers with frontend


npm i -D nodemon concurrently
    ==> nodemon for restart the server automatically
        concurrently for run frontend and backend at a time

npx create-react-app client
    ==> for initilaing react app
    ERROR :- You are running Node 12.22.10.Create React App requires Node 14 or higher.Please update your version of Node.
    SOLUTION :- https://www.digitalocean.com/community/tutorials/how-to-install-node-js-on-ubuntu-20-04

-------------------------------------
COMMANDS FOR RUNNING PROJECT
-------------------------------------

npm start ==> running project

npm  run server ==> running with nodemon package

npm  run both ==> to run frontend and backend
